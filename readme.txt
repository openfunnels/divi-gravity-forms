== Installation ==

Installing Gravity Divi can be done either by using the following steps:

1. Download the plugin via DiviCake.com, etc.
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
NOTE: Gravity Divi requires the Gravity Forms plugin to be installed and active.


Got questions? visit https://gravitydivi.com/ or https://divicake.com/author/CODECRATER/
